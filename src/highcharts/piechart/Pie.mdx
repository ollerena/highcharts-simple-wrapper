# Pie Chart

<PropsTable of={Pie} />

### Nested types

```js
type Point = {
	name: string,
	y: number,
	visible: boolean
};
```

```js
type Series = {
	/** Please entry the colors in HEX e.g. '#cfcfcf' or RGB e.g. 'rgb(11, 95, 255)' */
	colors: string[],
	data: ([string, number] | Point)[],
	outerRadius?: number,
	innerRadius?: number
};
```
